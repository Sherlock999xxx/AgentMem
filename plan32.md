```
                            ↓
┌─────────────────────────────────────────────────────────────────────┐
│                    三层存储架构                                        │
├─────────────────┬─────────────────┬─────────────────────────────────┤
│  Vector Database │  Entity Store    │   SQL Database                 │
```

#### Multi-Signal Retrieval (多信号检索)

```python
# 并行评分三个信号
retrieval_signals = {
    "semantic": vector_similarity_score,    # 语义向量相似度
    "keyword": bm25_score,                  # BM25关键词匹配
    "entity": entity_graph_score            # 实体图谱匹配
}
```
